# 🚀 GenAI ContentOps 2.0

## 🔥 Overview

GenAI ContentOps is a multi-agent AI platform that automates the entire enterprise content lifecycle — from creation to compliance, localization, distribution, and analytics.

## 🧠 Problem

Enterprise content creation is:

* Time-consuming
* Non-compliant
* Not localized for Indian audiences

## 💡 Solution

We built a **5-agent autonomous AI pipeline**:

1. 📝 Draft Agent → Generates content
2. 🔍 Compliance Agent → Ensures regulatory safety
3. 🌐 Localization Agent → Hindi + Hinglish
4. 📡 Distribution Agent → Multi-platform publishing
5. 🧠 Intelligence Agent → Analytics + feedback loop

## ⚙️ Tech Stack

* Python
* FastAPI
* Streamlit
* Multi-Agent Architecture

## 🚀 How to Run

### Backend

uvicorn main:app --reload

### UI

streamlit run dashboard.py

## 🎯 Demo

Enter a topic → Click **Run Pipeline** → See full AI workflow.

* ⏱️ Reduces content creation time by 70%
* 📊 Ensures compliance automatically
* 🌍 Supports Indian multilingual audience
